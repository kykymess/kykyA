libFun: boite à outil de différentes fonctions de transfert
ffNet: Réseau FeedForward - différents tests (and, or, xor & sinus)
ffNet_extended: La même mais avec sortie graphique
ffElman: Réseau SRN (Simple Recurrent Network) ou réseau de Elman
         permet de modéliser une mémoire dans un réseau (série temporelle)
ffRBoltzman : Restricted Boltzmann Machine, réseau à comportement probabiliste

test_digits : mise en oeuvre sur une base de chiffres manuscrits
              soit la base est récupérée depuis sklearn soit en local

Ces codes sont à visée pédagogique, pas d'optimisation, pas de manuel
autre que les cours / support de cours
